# 7S-02: STANDARDS
**Library**: simple_showcase
**Status**: BACKWASH (retroactive documentation)
**Date**: 2026-01-23

## Applicable Standards

### Web Standards
- **HTML5**: Semantic markup
- **CSS**: Tailwind CSS framework
- **JavaScript**: Alpine.js for interactivity
- **Responsive**: Mobile-first design

### HTTP Standards
- **Clean URLs**: No .html extensions
- **Redirects**: 301/302 for URL normalization
- **CORS**: Cross-origin handling
- **Rate Limiting**: 429 Too Many Requests

### Email Standards
- **SMTP**: RFC 5321 compliant
- **Security**: Input sanitization for injection prevention

## Design Patterns

1. **Page Pattern**: Deferred SSC_PAGE base class
2. **Section Pattern**: Reusable SSC_SECTION components
3. **Middleware Pattern**: Analytics, logging middleware
4. **MVC-like**: Pages generate HTML, server routes requests

## Frontend Stack

- **Tailwind CSS**: Utility-first styling
- **Alpine.js**: Reactive UI components
- **Lenis**: Smooth scrolling
- **Service Worker**: Offline caching
